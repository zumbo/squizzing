Squizzing is an online quizzing tool that includes the following functions:
- Users (players) can answer multiple choice questions.  
- A question consists of a short text and/or an image.
  For each question, there are four answer options, of which only one is correct.
  An answer option contains of a short text and/or an image.
- The user can choose one answer option. There is a timer that runs for 10s; if the user answers immediately, 
  they get 100 points, if they take 10s or longer, they get 50 points; for an intermediate time, the points are 
  interpolated linearly. If the answer is wrong, no points are awarded. 
- A round consists of 20-30 questions. Two rounds are played per week; there is only one active round at a time that can be played.
- After a player has played a round, the total score is calculated. The personal score and the scoreboard is displayed. 
- Optionally, there's an explanation for the question/answer, that is displayed after the player has answered the question.
- Administrators can upload questions. All questions of one round are contained in an Excel sheet or CSV file.
- Administrators can edit uploaded questions.
- Administrators can add or remove users.
